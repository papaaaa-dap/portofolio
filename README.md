# Portfolio — Neobrutalism

Personal portfolio website built with **Next.js**, **Framer Motion**, **Tailwind CSS**, and **Lenis** smooth scrolling.

## Sections

- Hero — typewriter effect, floating badges, CTA buttons
- About — bio, neobrutalism stat cards
- Experience — timeline
- Skills — icon grid
- Certificates — certificate cards
- Services — service offerings
- Projects — project showcase with modal detail
- Contact — contact form

## Tech Stack

| Tool | Purpose |
|------|---------|
| Next.js 14 | React framework |
| Framer Motion | Animations |
| Lenis | Smooth scrolling |
| Lucide React | Icons |
| Tailwind CSS | Styling |
| TypeScript | Type safety |

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```
